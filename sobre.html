Sobre

postado 18 maio 2021 Fonte: Psicologia Explica

Este site foi desenvolvido pelos alunos do curso de Análise e Desenvolvimento de Sistemas da faculdade Fametro com fins didáticos para a realização do INOVATECH.

O INOVATECH tem como proposta desenvolver um projeto que possa contribuir com ferramentas que ajudem, de certa forma, a sociedade. Para o desenvolvimento deste presente site, foi proposta a ideia de uma ferramenta que possa auxiliar as pessoas que foram afetadas psicologicamente pela pandemia de covid-19.

A ferramenta foi desenvolvida na plataforma da Google chamada DialogFlow, que é uma plataforma que simula a linguagem natural e compreende os padrões que as pessoas conversam, conseguindo dialogar com as pessoas através da interface de um chat, ele também tem um suporte para sistemas que tem interação com a voz.

Todo o corpo do site foi desenvolvido comas tecnologias da WEB, foi utilizado HTML 5 e CSS 3. HTML é uma linguagem de marcação de texto, nela podemos fazer a marcação das tags, adicionar imagens e textos na página, além de possuir varias tags para as mais variadas funções. O CSS é um mecanismo que ajuda na estilização das páginas da web, nele podemos adicionar várias formatações e estilos que deixam as páginas mais bonitas.
